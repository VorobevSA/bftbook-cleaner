# Security Policy

## Reporting a Vulnerability

Since this tool only processes local data, you can **publicly report** any security issues or bugs you find.

Please open a [bug report issue](https://github.com/your-username/CometBFT-Addrbook-Cleaner/issues/new?template=bug_report.md) and include:

- Description of the issue
- Steps to reproduce
- Expected vs actual behavior

We appreciate any security issues or bugs you discover!
